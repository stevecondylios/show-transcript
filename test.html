How to Test Your Extension

Step 1: Install the Extension

Make sure you've loaded the extension in developer mode:

  1. Open Chrome and go to chrome://extensions/
  2. Enable "Developer mode" in the top-right corner
  3. Click "Load unpacked" and select this folder

Step 2: Create the Extension Icons

Before testing on the Chrome Web Store, you'll need icons:

  1. Open icons/create_icons.html in your browser
  2. Right-click on each canvas and save as the appropriate filename in the icons folder
  3. Make sure you have icon16.png, icon48.png, and icon128.png

Expected Behavior

  • After the video loads, the transcript should automatically appear below the description
  • Clicking on timestamps should jump to that point in the video
  • Clicking the "Copy" button should copy the transcript text without timestamps
  • The extension should work in both light and dark modes

Troubleshooting

  • If the transcript doesn't appear, check if the video has a transcript available
  • Try refreshing the page if the extension doesn't load properly
  • Check the browser console for any error messages
  • Make sure your extension is properly loaded in chrome://extensions/
